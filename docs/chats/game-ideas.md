## Extracted game-oriented concepts
- `Learn-to-Earn`: complete AI/neuromorphic learning tasks and earn `$NEURO`.
- `Agent Companion`: Neo acts as in-game guide, tutorial voice, and challenge host.
- `Community Quests`: social tasks (sharing, onboarding, collaboration) tied to token rewards.
- `King of the Hill` motif: recurring competitive status/challenge framing that can map to seasonal ladders.
- `Virtuals`-style public terminal: transparent agent state/thought feed as a gameplay/social mechanic.
- `OnCyber gallery` progression: build and unlock showcase spaces as players complete milestones.
- Cyberpunk comic arcs (`Neo Awakens`, `The Resistance Upload`): reusable lore/quest chapters and event narrative.

## Candidate feature hooks for the main game
- Mission chains based on efficiency goals (reduce energy, improve throughput, stabilize network).
- Node-building puzzles tied to spiking behavior and event-driven processing.
- Seasonal narrative updates driven by comics/podcast episodes.
